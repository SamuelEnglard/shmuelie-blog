The goal of this post is to list all the cross platform UI frameworks for .NET/Mono I can find. I’ll update it with new ones as I find them. I am not recommending any of these over any others. I’m simply listing them.

Commercial:

  • Noesis: Support Windows, Windows Store, OSX, Linux, Windows Phone 8, Android, and iOS.
  • Xamarin.Forms: Supports iOS, Android, and Windows Phone 8.1
  • C#/XAML for HTML5: The Web so…

Closed Source

  • Empty Keys UI: Based on MonoGame, FNA, or SunBurn. OS support based on underlying engine.
  • NOV User Interface: Supports Windows and OSX.

Open Source

  • Ruminate MonoGame GUI: Supports Windows with beta support for OSX.
  • WPF Light: Supports Windows, Windows Store, OSX, Linus, iOS, Android, Windows Phone, and PlayStation 4.
  • Avalonia: Supports just Windows?
  • Perspex: UNIX, Windows, OSX.
  • ETO: Supports Windows, OSX, and Linux with iOS in beta.
  • XWT: Supports Windows, OSX, and Linux.

Hope someone finds this helpful!
