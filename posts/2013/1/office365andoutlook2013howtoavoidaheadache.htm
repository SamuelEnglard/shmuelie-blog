I recently switch to Office 365 for my email, for a variety of reasons. Hooking it up to Outlook 2013 has been a nightmare so I thought I’d share how to do it so you don’t go through it too.

  1. In the “Add Account” wizard select to configure the account manually and select an Exchange Account.
  2. Next log in to the Outlook Web App for your Office 365 email. From here we’re get all the information we need.
  3. Open the “About” page under the help menu.
  4. Copy the domain under “Host address”. Will look something like "https://bl2prd0812.outlook.com/owa" (so we want the bl2prd0812.outlook.com part).
  5. Back in Outlook set the server to the domain we just copied but add .mailbox before outlook.com (bl2prd0812.mailbox.outlook.com)
  6. Set the user name to your FULL email address (include the domain)
  7. Open the more settings. In there go to the last tab, “connection”.
  8. Check the “use HTTP proxy” checkbox.
  9. Open the proxy settings.
  10. Select ALL the checkboxes.
  11. For the proxy URL give the domain from before (bl2prd0812.outlook.com).
  12. For the principle name for certificates use “msstd:outlook.com” (without the quotes).
  13. Finally set the authentication method to basic.

And that should do it!
